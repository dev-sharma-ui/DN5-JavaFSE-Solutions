# Git Hands-on Lab 5

## Objective

Learn common Git maintenance and inspection commands.

---

## View Commit History

```bash
git log
```

---

## Compact Commit History

```bash
git log --oneline
```

---

## Show File Differences

```bash
git diff
```

---

## View Repository Status

```bash
git status
```

---

## Display Commit Details

```bash
git show <commit-id>
```

---

## Stash Current Changes

```bash
git stash
```

---

## View Stashed Changes

```bash
git stash list
```

---

## Restore Stashed Changes

```bash
git stash apply
```

---

## Remove Stash

```bash
git stash drop
```

---

## Delete Untracked Files

```bash
git clean -f
```

---

## View Git Configuration

```bash
git config --list
```
